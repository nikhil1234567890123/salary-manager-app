# Features List: Salary Manager

The Salary Manager app is packed with features designed to automate financial tracking and provide actionable insights.

## 1. Salary & Budget Setup
- **Configurable Income**: Set your monthly salary and fixed expenses (Rent, EMI, etc.).
- **Automatic Savings Target**: default 20% savings goal calculation.
- **Dynamic Daily Limit**: Calculates a "Safe Daily Spend" based on remaining days and balance.

## 2. Automated Expense Detection
- **SMS Reading**: Scans incoming SMS for transaction alerts (UPI, Bank, etc.).
- **Intelligent Parsing**: Extracts amount, merchant, and transaction type using advanced regex patterns.
- **Merchant Learning**: A mapping system that identifies merchants (e.g., Swiggy, Zomato) from cryptic transaction strings.
- **Duplicate Prevention**: Algorithms to ensure the same transaction isn't recorded twice.

## 3. Financial Analytics & Dashboard
- **Real-time Overview**: Instant view of total expenses, spendable money, and current savings.
- **Spending Trends**: Visual representation (charts) of weekly spending habits.
- **End-of-Month Prediction**: Smart forecasting to tell you if you'll meet your savings target.
- **Daily Usage Indicators**: Color-coded progress bars (Safe/Warning/Overspent).

## 4. Smart Suggestions
- **Personalized Insights**: Context-aware tips (e.g., "You're spending heavily on Food this week").
- **Savings Nudges**: Alerts when you are close to hitting your monthly budget.

## 5. Notification System
- **Immediate Alerts**: Instant notifications when a new expense is detected.
- **Overspending Warnings**: Proactive alerts when your daily spending exceeds the safe limit.
- **Local Sound & Vibration**: Premium alert feedback for financial events.

## 6. Report & Data Management
- **CSV Export**: Export your monthly spending data for external use (tax filing, personal records).
- **Monthly Reset**: Smooth transition into a new month with fresh budget calculations.
- **Secure Persistence**: All data is stored locally on the device for maximum privacy.

## 7. Security
- **App Lock**: Integrated biometric or PIN-based locking to protect financial data.
