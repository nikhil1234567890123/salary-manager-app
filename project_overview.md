# Project Overview: Salary Manager

Salary Manager is a premium, high-performance fintech mobile application designed to help users track their monthly income, manage fixed expenses, and maintain financial discipline through intelligent automation.

## Vision
The application aims to provide a "premium personal accountant" experience, combining sleek dark/gold aesthetics with proactive financial tools. Unlike traditional trackers, Salary Manager reduces manual entry by leveraging automatic SMS detection and smart predictive analytics.

## Core Value Proposition
- **Effortless Tracking**: Automatically detects UPI and Bank transactions from SMS notifications.
- **Financial Guardian**: Calculates "Safe Daily Spend" to ensure the user never overspends their monthly budget.
- **Data-Driven Insights**: Provides end-of-month predictions and personalized financial suggestions.
- **Privacy First**: Secure local data management with optional app-lock features.

## Target Audience
- Salaried professionals and students receiving monthly stipends.
- Users looking for a low-friction way to categorize and monitor their spending.
- Individuals who need proactive alerts to stay within their budget.

## Technology Stack
- **Framework**: Expo (React Native) with SDK 54.
- **Language**: TypeScript for robust type safety.
- **Styling**: NativeWind (Tailwind CSS) for modern, responsive UI.
- **Animations**: React Native Reanimated for premium micro-interactions.
- **Navigation**: Expo Router (File-based routing).
